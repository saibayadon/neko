# 馃悎鈥嶁瑳
## Node.js & TypeScript

### Defaults
- **TypeScript 5.0.2**

### How to run locally
- `yarn install`
- `yarn start`

### Build for production
- `yarn build` -> Will output a dist folder

### Linting
**ESLint** (Using TypeScript defaults)
- `yarn lint` -> Will lint all code under `src` folder.
